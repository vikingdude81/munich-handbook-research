# Review Guide for Distilled Outputs

To review the latest distilled outputs in `distill/necro/` (e.g., chunk_01.json, chunk_25.json), follow these steps:

1. Open each JSON file.
2. Inspect for:
   - Spirit names
   - Ranks
   - Functions
   - Appearances
   - Legion counts
   - Conjuration methods
   - Experiment references
   - Page/folio numbers
   - Raw quotes presence

Ensure all extracted data is accurate and consistent across files.
