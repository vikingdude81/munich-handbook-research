# Spirits and Experiments Analysis

## Spirits Schema

- `name` (string)
- `type` (string)
- `rank` (enum: "demon", "king")
- `legion_count` (number)

## Experiments Schema

- `date` (date)
- `location` (string)
- `method` (string)

## Notes

- `legion_count` is numeric.
- `rank` is always one of ["demon", "king"].
