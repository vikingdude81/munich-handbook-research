(a) The expected schema for the distilled JSON should include at least the following keys:
- `spirit_name`: Name of the spirit.
- `rank`: Rank or category of the quote (e.g., wisdom, humor).
- `raw_quote`: The actual quote text.

(b) Distillation should produce one spirit per file to ensure that Phase 2 parsing is schema-compliant and each spirit's data can be processed independently.
